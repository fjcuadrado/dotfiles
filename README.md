# dotfiles

This repository contains my configuration files (aka dotfiles) for some of the software that I use.

You can use these files cloning this repository into, for example, the `~/.config` directory and running the given `Makefile`.

  * Install all my dotfiles: `make`
  * Install a given target, for example `vim`: `make vim-install`
  * Uninstall all my dotfiles: `make clean`
  * Uninstall a given target, for example `vim`: `make vim-uninstall`
